# Ghost
Python program that sanitizes the following fields of metadata in PDF files:
* /Author
* /Producer
* /Title  
* /Subject
* /Creator
* /Keywords
* /CreationDate
* /ModDate
* /Trapped
* /PTEX.Fullbanner

## Python
The used python version is:

```
version = 3.12.3
```

## Install requirements

```bash
pip install -r requirements.txt
```
