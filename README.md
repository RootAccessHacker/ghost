# Ghost
Python program that sanitizes the following fields of metadata:
* /Author
* /Producer
* /Title  
* /Subject
* /Creator
* /Keywords
* /CreationDate
* /ModDate
* /Trapped
* /PTEX.Fullbanner

python --version
